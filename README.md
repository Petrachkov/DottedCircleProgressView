## Requirements

Swift 4.0, iOS version >= 8.0

## Installation

DottedCircleProgressView is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod "DottedCircleProgressView"
```
